<summary><strong>🏭 Steel Wizard AI — Industrial AI Copilot for Predictive Maintenance</strong></summary>

<br>

**Overview**

Steel Wizard AI is an industrial maintenance intelligence platform built for steel manufacturing environments. It combines predictive maintenance, reliability engineering, asset health monitoring, and AI-assisted decision-making into a single production-deployed system.

**Live Demo:** [ai-maintainance-wizard-core.lovable.app](https://ai-maintainance-wizard-core.lovable.app)

**Architecture**

```
Critical Industrial Assets
         │
         ▼
  Supabase (PostgreSQL)
         │
         ▼
  Steel Wizard AI Platform
         │
  ┌──────┼────────┬──────────┐
  ▼      ▼        ▼          ▼
Assets Alerts  Spares    Reports
         │
         ▼
  AI Maintenance Wizard
  (Natural Language Interface)
         │
         ▼
  Recommendations & Decisions
```

**Key Features**
- Natural language interface for maintenance queries — diagnosis, root cause analysis, RUL prediction, spare recommendations
- Reliability Command Center with plant health scoring and failure probability tracking
- Asset Intelligence with health scores, failure probability, and remaining useful life (RUL)
- Alert prioritization using severity, business impact, and risk classification
- Automated report generation: daily reliability, shift handover, shutdown planning, executive summaries

**Technology Stack**

`TypeScript` `React` `Tailwind CSS` `ShadCN UI` `Supabase` `PostgreSQL` `Lovable`

**Engineering Highlights**
- Full-stack TypeScript application with a Supabase PostgreSQL backend
- Modular component architecture covering assets, alerts, spares, and reporting domains
- AI wizard integrated as a natural language decision support layer over structured operational data
- Deployed to production with a live public demo

<details>
<summary><strong>🧠 HireMind AI Recruiter — Intelligent Candidate Discovery Engine</strong></summary>

<br>

**Overview**

HireMind AI is a candidate ranking and discovery platform built for the India Runs Hackathon. It processes up to 100,000 candidate profiles using a multi-factor explainable scoring engine — going beyond keyword matching to evaluate technical depth, production experience, behavioral signals, and risk indicators.

**Architecture**

```
Candidate Dataset (100K profiles)
         │
         ▼
  Candidate Intelligence Engine
         │
  ┌──────┼──────────┬─────────┐
  ▼      ▼          ▼         ▼
Tech   Production Behavioral Risk
Fit    Experience  Signals   Detection
         │
         ▼
  Ranking Engine → Top-100 Shortlist → CSV Export
```

**Ranking Framework**

| Factor | Weight |
|---|---|
| Technical Fit | 40% |
| Production Experience | 20% |
| Behavioral Signals | 15% |
| Experience Fit | 10% |
| Domain Evaluation | 10% |
| Availability | 5% |

**Key Features**
- Multi-dimensional candidate scoring with fully explainable outputs per candidate
- Behavioral intelligence layer: GitHub activity, recruiter response rate, offer acceptance rate, profile completeness
- Trap detection system that identifies keyword-stuffed or misleading profiles with risk penalties
- React + TypeScript frontend with executive dashboard, candidate explorer, behavioral analysis, and comparison views
- Python/FastAPI backend with streaming ingestion and vectorised scoring using pandas/numpy

**Technology Stack**

`TypeScript` `React 19` `TanStack Start` `Tailwind CSS` `Recharts` `Python` `FastAPI` `Pandas` `NumPy`

**Engineering Highlights**
- TypeScript ranker and Python ranker use identical scoring formulas — consistent dual-implementation
- Scale-out path documented: Sentence Transformers → FAISS ANN retrieval → learning-to-rank → LLM explanations
- REST API endpoints: `/ingest`, `/rank`, `/candidate/{id}`, `/compare`, `/stats`

[![Repo](https://img.shields.io/badge/GitHub-HireMind--AI--Recruiter-181717?style=flat-square&logo=github)](https://github.com/karthikeya-Hub12/HireMind-AI-Recruiter)
